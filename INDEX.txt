TROUBLESHOOTING
---------------
Issue: Agent gets 0 score
Fix: Check for syntax errors in evolved code

Issue: Evolution is slow
Fix: Reduce num_games (default: 5) or max_steps (default: 3000)

Issue: No improvement
Fix: Ensure multiple games per eval, try smaller code changes

Issue: Want to see what's happening
Fix: Set render=True in evaluator, or run demo.py 2
